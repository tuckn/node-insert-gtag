Node.js: insert-gtag

Node.js CLI to insert a Google Analytics code (gtag.js) into your HTML files.

Installation

npm install -g @tuckn/insert-gtag

Usage

analytics

$ insert-gtag analytics --help

Usage: insert-gtag analytics [options] <dirPath> <trackingId>

Insert Google Analytics code (gtag.js) in html files

Options:
  -V, --version           output the version number
  -E, --ext <name>        A extension of files to be inserted. Default: .html, .htm
  -X, --exclude <glob>    [_\\-.]cache\\d+
  -I, --indents <number>  Number of indenting white spaces. Default: 4
  -A, --adding-tag <code> @todo
  -h, --help              display help for command

The below is example.

$ insert-gtag analytics "D:\HtmlDir" "UA-9999-9"
